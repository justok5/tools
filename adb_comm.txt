adb shell   启动Shell模式

adb kill-server 结束adb服务
adb devices	列出联机设备
adb version 版本号

adb install <APK> 安装程序
adb install -r <APK> 安装程序但保留原有数据
adb uninstall <APK> 删除程序
adb uninstall -k <APK> 删除程序但保留数据

adb push <local> <remote>  拷贝到手机
adb pull <remote> [<local>] 拷贝到电脑

adb  reboot bootloader  重启手机到fastboot模式

---------- 实例 -----------

$ su	提权
# mount -o remount /system	修改system为读写权限(或者用 adb remount)
# cd /system/app
# rm adv.apk
# chmod 744 tool.apk
